Use Liststrap to integrate Mailchimp signup forms into your WordPress site without trouble.

== Description ==

Liststrap is a free and simple to use mailchimp plugin which You can use to integrate Mailchimp signup/newsletter forms into your WordPress site.
You can insert the shortcode into Posts, pages, sidebars, and other widgetized areas to display Mailchimp forms.
Your Mailchimp API Key connects your website to your account, allowing you to access all of your list information.
Statistics for your lists can be accessed directly from your dashboard. 

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload the 'liststrap' folder to the directory '/wp-content/plugins/'.
2. Activate the plugin using the 'Plugins' menu in WordPress.
3. You can adjust the necessary settings using your WordPress admin panel in 'ListStrap' section.

== Frequently Asked Questions ==

= Do I Need a mailchimp Account? =

Yes, you must have a Mailchimp account and at least 1 list set up in order to use the plugin. 

= How does Mailchimp account connects to my site using Liststrap?  =

Liststrap uses an API Key to allow your site to communicate with your mailchimp account. You will have to enter your API key into the plugin settings of your site to get connected with your account and use the form.

= How do I create a Mailchimp API key? =

Step-by-step instructions:

1. Click your profile name to expand the Account Panel, and choose Account.
2. Click the Extras drop-down menu and choose Mailchimp API keys.
3. Copy an existing API key or click the Create A Key button.
4. Name your key descriptively, so you know what application uses that key.
